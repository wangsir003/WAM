# 日志控制台优化更新

## ✅ 已完成的优化

### 问题描述
- 日志控制台在页面最下方
- 运行项目时需要滚动才能看到日志
- 影响实时监控体验

### 解决方案

#### 1️⃣ 可展开/收起的日志面板

**默认状态（收起）：**
- 高度：200px
- 显示最近的日志
- 不占用过多空间

**展开状态：**
- 高度：600px
- 显示更多日志内容
- 一键快速切换

#### 2️⃣ 一键展开按钮

**位置：** 日志卡片标题右侧

**功能：**
- 收起状态：显示"展开 ↑"按钮
- 展开状态：显示"收起 ↓"按钮
- 点击快速切换
- 展开时自动滚动到最新日志

#### 3️⃣ 日志数量徽章

**显示：** 实时日志标题旁边

**功能：**
- 实时显示当前日志总数
- 最多显示 999+
- 提示用户有新日志

#### 4️⃣ 内容区可滚动

**优化：** 主内容区添加滚动
- 所有卡片（项目详情、设备、工具箱、日志）可独立滚动
- 不会因为内容过多而隐藏日志

## 🔧 技术实现

### 1. 状态管理
```javascript
const isExpanded = ref(false); // 默认收起
```

### 2. 动态高度
```css
.log-console-card {
  max-height: 200px; /* 收起 */
  transition: max-height 0.3s ease;
}

.log-console-card.log-expanded {
  max-height: 600px; /* 展开 */
}
```

### 3. 展开功能
```javascript
function toggleExpand() {
  isExpanded.value = !isExpanded.value;
  if (isExpanded.value) {
    // 展开时自动滚动到底部
    nextTick(() => scrollToBottom());
  }
}
```

### 4. 日志徽章
```vue
<a-badge :count="logs.length" :overflow-count="999" />
```
