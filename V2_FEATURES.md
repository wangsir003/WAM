### 4. 开发者快捷工具箱

**新增组件：DevToolbox.vue**

包含以下功能：

#### 4.1 批量快捷操作
- ✅ **批量截屏**：
  - 执行 `adb shell screencap` 截屏
  - 通过 `adb pull` 保存到桌面
  - 文件命名：`项目名_设备ID_时间戳.png`
  - 自动删除设备上的临时文件
  - 成功后弹出通知

- ✅ **一键清理缓存**：`adb shell pm clear [包名]`
- ✅ **一键强杀应用**：`adb shell am force-stop [包名]`
- ✅ **一键卸载应用**：`adb uninstall [包名]`

所有操作均支持**批量执行**（对所有选中设备）

#### 4.2 Logcat 实时监控
- ✅ 设备选择下拉框
- ✅ 开启/停止监控按钮
- ✅ 优先过滤当前应用的 PID
- ✅ **自动高亮关键错误**：
  - `Exception`、`NullPointerException`、`CRASH`、`FATAL`、`ERROR` → 红色
  - `WARN` → 黄色
- ✅ 实时监控状态指示（绿色闪烁点）
- ✅ 应用退出时自动清理 logcat 进程

## 🚀 使用说明

### 1. Release 打包
1. 选择项目
2. 点击"一键 Release 打包"
3. 等待编译完成
4. Release APK 文件夹自动打开

### 2. WiFi 连接
1. 在手机上执行：`adb tcpip 5555`
2. 获取手机 IP 地址
3. 在输入框输入：`192.168.1.100:5555`
4. 点击"连接"
5. 设备列表自动刷新

### 3. 批量截屏
1. 勾选多个设备
2. 点击"批量截屏"
3. 截图自动保存到桌面

### 4. Logcat 监控
1. 选择监控设备
2. 点击"开启监控"
3. 查看实时日志（错误自动红色高亮）
4. 点击"停止监控"结束

## ⚠️ 注意事项

1. **Release 编译**需要项目已配置签名（keystore）
2. **WiFi 连接**需要手机和电脑在同一网络
3. **Logcat 监控**会持续运行直到手动停止
4. **批量截屏**文件保存在用户桌面
