## 🔧 技术实现

### 1. 状态管理（stores/project.js）

```javascript
const isRunning = ref(false);

function setRunning(running) {
  isRunning.value = running;
}
```

### 2. 布局结构（App.vue）

```vue
<div class="content-wrapper">
  <!-- 功能区：默认100%，运行时50% -->
  <div :class="['function-area', { 'full-width': !isRunning }]">
    <ProjectDetail />
    <DeviceManager />
    <DevToolbox />
  </div>

  <!-- 日志区：仅运行时显示，占50% -->
  <div v-if="isRunning" class="log-area">
    <LogConsole />
  </div>
</div>
```

### 3. 日志过滤（LogConsole.vue）

```javascript
function isImportantLog(line) {
  const importantKeywords = [
    'BUILD', 'Task :', 'Success', 'Failed',
    'Error', 'Warning', 'Exception',
    '安装', '启动', '✓', '✗', '⚠',
    'Installing', 'Launching', 'CRASH'
  ];
  
  return importantKeywords.some(keyword => 
    line.includes(keyword)
  );
}
```

### 4. 运行状态触发（ProjectDetail.vue）

```javascript
async function buildAndInstall() {
  projectStore.setRunning(true); // 显示日志区
  
  try {
    // 编译和安装逻辑
    await window.electronAPI.buildProject(...);
  } finally {
    // 保持日志区显示，用户手动关闭
  }
}
```
